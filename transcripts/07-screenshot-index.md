# Screenshot Index

This file maps every test ID to its screenshot file. Use it to find visual evidence for any test in the evaluation.

All screenshots show the AI Assist response panel scrolled to the **top** of the answer, so they include the thinking phase, the Management API Query tool calls, and the beginning of the rendered response. For long responses, the bottom half may be cut off — the full text is always available in the corresponding response transcript file.

## File naming convention

- Pattern: `test-<TEST_ID>-top.png`
- Examples: `test-A001-top.png`, `test-B005-top.png`, `test-J001-top.png`, `test-CONV-T1-top.png`, `test-CONV007-T2-top.png`, `test-SLOPPY-001-T6-top.png`, `test-MULTI-001-T4-top.png`

When the team asks Claude "show me the screenshot for X", Claude can reference the matching file from Project Knowledge.
